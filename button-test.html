Testing Instructions

  1. Go to Appearance → Customize → Button Styles
  2. Toggle the checkboxes for each button type:
    • Outline Style - Changes from filled to outlined
    • Rounded Corners - Changes from rounded to square
  3. The buttons on this page should update in real-time when you change the settings
  4. Verify hover states work correctly

Default Settings:

  • Primary: Filled, Rounded
  • Secondary: Outline, Rounded
  • Tertiary: Filled, Rounded
